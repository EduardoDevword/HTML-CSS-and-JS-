Objetivo do trabalho:

O projeto tinha como objetivo a utilização de python para desenvolver um programa que utilizasse "Tkinter" uma biblioteca da linguagem que produz uma tela na maquina do usuario onde se poderia por conteudos e funcionalidades

Metodos usados para crição do projeto:

  • Python 3
  • Biblioteca Tkinter
  • Banco SQL Server 2019
